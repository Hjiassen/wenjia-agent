# Changelog

All notable changes to this project will be documented here.

The format follows the spirit of Keep a Changelog, and this project uses
semantic versioning once tagged releases begin.

## [0.1.0] - 2026-07-03

### Added

- OpenAI Agents SDK based multi-agent workflow for Chinese metaphysics scenarios.
- Deterministic BaZi calculation core with tool-first Agent access.
- Specialized Agents for profile collection, fortune analysis, relationship
  analysis, naming, and tool explanation.
- Structured report schemas, prompt-as-code templates, input guardrails, session
  memory, long-term browser-client memory, and local JSONL tracing.
- FastAPI API-only web backend with JSON and SSE endpoints.
- React + TypeScript + Ant Design X web frontend with chat history, profile
  panel, long-term memory management, and run-flow visualization.
- CLI examples, offline eval contracts, unit tests, and development docs.
- Ubuntu helper scripts for preview-mode development serving and static Nginx
  publishing.
- GitHub project templates, CI, Dependabot, Docker/Compose, security policy, and
  code of conduct for open-source release.
