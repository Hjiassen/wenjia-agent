# Code of Conduct

## Our Pledge

We want `wenjia-agent` to be a welcoming project for people improving Chinese
metaphysics tooling, Agent orchestration, prompts, web UX, deployment, and
documentation. We expect contributors to be respectful, patient, and precise.

## Expected Behavior

- Be kind and professional, especially when disagreeing.
- Assume good intent, but be concrete about technical risks.
- Keep discussions focused on the project and the user impact.
- Respect privacy; do not post API keys, private birth data, local databases,
  trace files, or personal contact details.
- Use responsible language around metaphysics content and high-stakes topics.

## Unacceptable Behavior

- Harassment, threats, insults, or sustained personal attacks.
- Posting someone else's private information.
- Spam, off-topic promotion, or repeated low-effort disruption.
- Encouraging unsafe use of the project for medical, legal, financial, mental
  health, or other high-stakes decisions.

## Enforcement

Maintainers may edit, hide, or remove comments and issues; close discussions;
or block users when needed to keep the project healthy.

If you see a conduct issue, open a minimal issue or contact a maintainer through
an available private channel. Do not include sensitive personal data in public
threads.
