# Agent 策划书

## 一句话介绍

`wenjia-agent` 是一个面向开源社区的中文命理 Agent 框架：用确定性八字工具保证基础命盘可靠，用 OpenAI Agents SDK 组织可扩展的命理对话与报告生成能力。

## 为什么做

传统命理软件通常把排盘、解释、用户系统、付费体系和运营后台耦合在一起，开源协作和二次集成成本高。`wenjia-agent` 选择只做 Agent 层，把用户、支付和商业化系统交给外部产品，从而更适合作为开源基础项目。

## 产品原则

- 确定性计算和生成式解释分离。
- 命盘字段由代码计算，Agent 只负责解释、组织和追问。
- 默认中文体验，适合中文命理语境。
- 不做恐吓式、宿命式表达。
- 保持轻量，方便嵌入已有产品。

## Agent 版图

| Agent | 作用 | 输出 |
| --- | --- | --- |
| WenjiaMainAgent | 主控路由与任务拆分 | 自然语言或 handoff |
| ProfileAgent | 收集出生资料、排盘 | 基础命盘摘要 |
| FortuneAgent | 命格综合分析 | `FortuneReport` |
| RelationshipAgent | 合盘与关系沟通建议 | `RelationshipReport` |
| NamingAgent | 起名策略与名字建议 | `NamingReport` |
| MysticToolsAgent | 工具查询、字段解释、参数排障 | 自然语言说明 |

## 开源价值

- 开发者可以直接复用八字工具和 Agent 编排。
- 社区可以围绕 prompt、工具、报告 schema 贡献。
- 上层产品可以保留自己的用户、支付和内容策略。
- Apache-2.0 许可证方便商业产品集成。

## 合作方式

- 核心维护者负责架构边界、发布节奏和质量门禁。
- 命理贡献者负责术语、解释框架、prompt 审校。
- 工程贡献者负责工具扩展、测试和示例集成。
- 产品贡献者负责使用案例、体验反馈和文档补充。
